Web MIDI API Shim

This is a polyfill using the Jazz NPAPI MIDI plugin to implement the Web MIDI API on Mac and Windows.

Download this project as a .zip file Download this project as a tar.gz file

To use the Web MIDI API Shim

Just include "<script src='http://cwilso.github.com/WebMIDIAPIShim/WebMIDIAPI.js'&rt;</script&rt;" in your HTML source.
